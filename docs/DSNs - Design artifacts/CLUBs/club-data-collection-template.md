## Common Data Patterns

### Email Patterns

- `info@[clubdomain].com`
- `contact@[clubdomain].com`
- `office@[clubdomain].com`
- `[clubabbrev]@[clubdomain].com`

### Phone Patterns

- US: `1[area_code][number]` (e.g., `13138221853`)
- Canada: `1[area_code][number]` (e.g., `15191234567`)
- Extensions: `[number];[extension]` (e.g., `13138221853;105`)

### Address Patterns

- US: `[number] [street] [street_type]`, `[city]`, `[state] [zip]`
- Canada: `[number] [street] [street_type]`, `[city]`, `[province] [postal_code]`

### Description Structure

1. Opening: Founding year and founders (if known)
2. Location: Current location and history of locations
3. Features: Key amenities, facilities, harbors
4. Significance: Notable events, races, historical importance
5. Closing: Current status or notable achievements
